## Требуется php весрии 8.0 и выше.

- Запускаем сервер через php -S 127.0.0.1:8000
- Файл бекап заливаем в базу
- Изменяем авторизационные данные базы в .env
- Открываем в браузере http://127.0.0.1:8000/index.html
